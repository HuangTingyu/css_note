# bacra-ui项目介绍

主要CSS代码 —— `bacra-ui\src`

demo —— `bacra-ui\demo`

### demo目录

- `flex_layout.html` —— flex布局仿微信
- `icon.html` —— 字体

字体图标主要引用 —— `fontawesome`

使用方式 —— 

您可以将Font Awesome图标使用在几乎任何地方，只需要使用CSS前缀 `fa` ，再加上图标名称。 Font Awesome是为使用内联元素而设计的。我们通常更喜欢使用 `<i>` ，因为它更简洁。 

- `layout.html` —— 简单布局
- `input.html` —— 表单
